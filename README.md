# 📊 CSV Data Analyzer App

An interactive web application built using **Streamlit** that allows users to upload CSV files and perform quick data analysis with ease.

🔗 Live App: https://csv-analyzer-app-eputuxkd8qww2kqlzvigb2.streamlit.app/  
🔗 GitHub Repo: https://github.com/nazilanazer2002-hue/csv-analyzer-app

---

## 🚀 Features

- 📁 Upload CSV files instantly
- 📊 View dataset (rows & columns)
- 🔍 Data summary and statistics
- 🧹 Handle missing values
- 📈 Basic data visualization
- 🔢 Perform calculations easily

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit  
- Pandas  
- NumPy
